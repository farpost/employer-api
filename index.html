API для работы с вакансиями и откликами на вакансии

Просмотр списка опубликованных вакансий работодателя

Возвращает список идентификаторов опубликовынных вакансий и ограниченный набор полей. Вакансии будут отсортированы по дате изменения. Максимальное число возвращаемых элементов — 100. Если запрос выполняется от лица дочернего кабинета, вернутся только вакансии, назначенные на него.

Authorizations:
OauthToken
query Parameters
modifiedAfter
string <date-time>

Возвращать вакансии с датой изменения от указанной даты. Если не передан — вернутся все вакансии. Дата в формате ISO 8601 с часовым поясом, например 2026-07-10T14:30:00+03:00.

cursor
string

Курсор из предыдущего запроса для получения следующей страницы

Применение действия к отклику

Authorizations:
OauthToken
path Parameters
action
required
string (VacancyResponseActionCode)
Enum: "view" "accept" "invite" "reject" "markAsContacted"

Применяемое действие. Получить список возможных действий можно, используя метод GET /api/job/dict/vacancy-response-actions

vacancyResponseId
required
integer

Идентификатор отклика на вакансию

Request Body schema: application/json
optional
message
string

Текст сообщения, которое будет отправлено соискателю после применении действия к отклику. Требуется для следующих действий:

  • accept
  • invite
  • reject

Если не передан, будет использован текст из шаблона, настроенного в личном кабинете.
